<포트폴리오를 만들면서 생긴 고민들>
1. 어떤 3Match Puzzle 게임을 만들어야 하는지? => 애니팡 1 시리즈로 하기로 결정.

2. 애니팡 1 시리즈의 어떠한 기능들까지 구현해야 하는지?
	ㄱ. 스테이지 데이터들을 직렬화&역직렬화 해야 하는지? => 스테이지 데이터를 없애기로 결정.
	ㄴ. 효과음이나 배경음을 넣어야 하는가? => 리소스 부재로 임시 후순위로 결정.
	ㄷ. 블럭의 형태나 종류는 몇 가지로 제한 하는지? => 다섯 가지 색상으로 구성하기로 결정.
	ㄹ. 블럭을 이동할 때 플레이어의 인풋 기준을 무엇으로 하는지? => 애니팡 1 시리즈를 플레이 해본 결과. 터치 'Began'할 때 좌표 기준으로 좌표에 해당하는 블럭이 활성화 되며, 터치 'Moved' 상태에서 (Moved 좌표 - Began 좌표)의 크기가 일정 이상이 될 경우, (Moved 좌표 - Began 좌표)의 각 x, y 중 절대값이 큰 쪽으로 블럭 스왑하기로 결정.
	ㅁ. 해상도 대응 필요한지? => 임시 방편으로 하기로 함.
	ㅂ. 매칭 우선 기준이 세로가 먼저인지 아니면 가로가 먼저인지 아니면 둘다 인지? => 임시로 둘 다 하기로 함.
